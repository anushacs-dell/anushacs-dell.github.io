Welcome to the ZOO-Project Trac


ZOO is a WPS (Web Processing Service) open source project released under the MIT/X-11 license. It provides an OGC WPS compliant developer-friendly framework to create and chain WPS webservices.

The official homepage of the project is ​http://zoo-project.org.

The ZOO Trac is hosting ZOO Project source code in a community oriented system. Please feel free to Register.
